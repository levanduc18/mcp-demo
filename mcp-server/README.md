# MCP Todo Server

## Installation

```bash
cd mcp-server
yarn install
```

## Running the Server

```bash
yarn start
```

Server will run on `http://localhost:3000/mcp`

## MCP Tools

The server exposes four MCP tools:

- `todo_create` - Create a new todo
- `todo_list` - List all todos
- `todo_update` - Update a todo
- `todo_delete` - Delete a todo
