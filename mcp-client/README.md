# MCP Todo Client

## Installation
```bash
cd client
npm install
```

## Running the Client
```bash
npm run dev
```

Client will run on `http://localhost:8080`

## Building for Production
```bash
npm run build
npm run preview
```

## Features

- Create todos with title and description
- List all todos
- Edit todos inline
- Delete todos
- Mark todos as completed
- Real-time sync with MCP server
- Clean and minimal UI
