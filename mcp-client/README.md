# MCP Todo Client

## Installation

```bash
cd mcp-client
yarn install
```

## Running the Client

```bash
yarn dev
```

Client will run on `http://localhost:8080`

## Building for Production

```bash
yarn build
yarn preview
```

## Features

- Create todos with title and description
- List all todos
- Edit todos inline
- Delete todos
- Mark todos as completed
- Real-time sync with MCP server
- Clean and minimal UI
